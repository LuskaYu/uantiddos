

# 服务日志管理


![](/images/uadssp/opintro/service_log.png)

基本信息

| 项目       | 描述                              |
| -------- | ------------------------------- |
| 资源ID     | 抗D服务包的唯一标识，当出现问题，需要定位和排查的时候请提交资源ID |
| 事件类型     | 操作的事件类型，比如联动事件和防护事件等                   |
| 操作     | 对防护包的操作内容，比如手动激活，手动切换cname指向高防等                    |
| 操作对象 | 操作的规则名称                   |
| 详情     | 操作的详细信息，比如开启防护，cname解析等           |
| 操作时间 | 操作的具体时间，时间精确到秒 |
